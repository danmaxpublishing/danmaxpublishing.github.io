Theming the UI

The entire photo mode UI (every panel, control, label, animation and sound) restyles from a single PhotoModeUITheme asset. Swap the asset to ship a new look.

Reskin in three steps¶

  1. Duplicate the shipped PhotoModeUITheme.asset (or create a fresh one via Assets > Create > Photo Mode Pro > UI Theme).
  2. Change colors, sprites, fonts, motion and audio on the duplicate.
  3. Assign it to the Theme field on the PhotoModeUIController component of your photo mode UI prefab.

The theme is applied at bind time (every time the UI spawns), so the next time photo mode opens, the whole panel wears your skin. At runtime you can also set PhotoModeUIController.Theme and call ApplyTheme() to restyle a live panel.

What the theme controls¶

Colors¶

Field Used for
panelColor Main panel background.
controlColor Slider/button/toggle backgrounds.
accentColor Fills, highlights, the selected tab.
textColor Primary labels.
dimTextColor Hints and value readouts.

Sprites and icons¶

panelSprite, controlSprite and handleSprite override the builder defaults (null keeps them). The generated 9-sliced sprites (roundedPanelSprite, roundedControlSprite, glowSprite, knobSprite, ringSprite) and the category icons (cameraIcon, effectsIcon, presetsIcon, captureIcon, snapIcon) can each be replaced with your own art.

Typography¶

tmpFont sets the TextMeshPro font asset for all labels (null falls back to the TMP default). tmpOutlineMaterial is an optional TMP material with a black outline/underlay used by scene-floating text (hint chips, the toast filename) so it stays readable over bright surfaces. A legacy font field exists as a fallback for non-TMP text.

TMP Essential Resources

The UI uses TextMeshPro. Import the essentials once via Window > TextMeshPro > Import TMP Essential Resources if your project has not already (the demo builder and the Setup Wizard handle this for you).

Motion¶

Field Default Meaning
enableMotion true Master switch for all UI motion.
hoverScale 1.05 Control scale while hovered/selected.
pressScale 0.95 Control scale while pressed.
punchAmplitude 0.08 Release punch wobble strength.
transitionDuration 0.25 s Master duration for panel/category transitions.

UI audio¶

Field Meaning
enableUiAudio Master switch for UI sounds.
uiVolume 0–1 volume for all UI sounds.
hoverClip / clickClip / tabClip Hover (or gamepad selection), click/submit, category change.
toggleOnClip / toggleOffClip Toggle switches.
shutterClip Played with the capture flash.
uiMixerGroup Optional AudioMixerGroup; null plays straight to the listener.

Kill switches

For a completely static, silent UI, set enableMotion and enableUiAudio to false. No other changes needed. Or keep motion and drop your own recorded sounds into the clip slots to replace the generated ones.

Scope¶

The theme restyles the photo mode panel and its floating chrome (toolbar, hint chips, toast). It does not touch your game's own UI, and gameplay HUD canvases are only ever hidden/shown by photo mode, never restyled (see Capturing photos for the visibility toggles).
