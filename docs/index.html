Photo Mode Pro

AAA-style, player-facing photo mode for Unity: pause the game, fly a free camera, grade the image, apply filter presets and capture supersampled screenshots — with a reskinnable, gamepad-ready UI.

Unity 2021.3 LTS or newer · URP, HDRP and the Built-in Render Pipeline.

What you get¶

  • One-component integration — drop the configured prefab into a scene (or use the Setup Wizard's Add Photo Mode To Scene button) and press your toggle key.
  • Free camera with fly and orbit modes, sprint, roll, FOV zoom, optional collision and a configurable wander radius.
  • Image adjustments in natural units — exposure (EV), contrast, saturation, temperature, vignette, film grain and bokeh depth of field with a perceptual f-stop slider.
  • Six filter presets (Clean, Noir, Vivid, Warm, Cool, Vintage) plus your own preset assets and an automatic Custom state.
  • Supersampled capture at 1×/2×/4× with async GPU readback, PNG output, optional transparent background, thumbnail toast and a capture event for your own pipelines.
  • All three render pipelines — URP, HDRP and Built-in (Post Processing Stack v2 or a minimal fallback) — through automatic backend detection, with controls hiding themselves when the active pipeline can't apply an effect.
  • Both input systems — the new Input System, the legacy Input Manager, or Both — keyboard + mouse and gamepad, with full UI navigation.
  • Zero per-frame GC allocations while open and idle or dragging sliders, and no per-frame work while closed. IL2CPP + high stripping verified.
