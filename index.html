The standard

What every DanMaxPublishing asset promises

Real documentation

The full manual is hosted here: quick start, concepts, how-tos, per-pipeline notes, and troubleshooting seeded from the real bug history. Every public API member carries XML docs. Check it before you buy; it's the same manual you get after.

Tested like production code

Every release has to pass EditMode and PlayMode suites first: GC allocation gates, state-restoration asserts, leak gates. Releases are tested on Unity 2021.3, 2022.3 and Unity 6, including an IL2CPP build with high stripping.

Support from the developer

Questions go to the person who wrote the code, not a ticket queue. Email or GitHub issues both work, and a built-in self-test writes a report you can attach. How to get help.
