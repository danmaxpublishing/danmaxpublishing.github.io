Platform & Pipeline Support Matrix

Unity versions¶

Unity Status
2021.3 LTS Supported (verified, URP 12)
2022.3 LTS Supported (verified, URP 14)
6000.x Supported (verified, URP 17 / HDRP 17 / Built-in)

Render pipelines¶

Pipeline Status
URP 12 / 14 / 17 Fully supported
HDRP 12+ Fully supported (no transparent capture; see Per-Pipeline Notes)
Built-in + Post Processing Stack v2 Fully supported
Built-in without PPv2 Supported with the minimal fallback backend (exposure, contrast, saturation, vignette)
Custom SRPs Not supported (the fallback backend and Standard-shader demo materials are used; no claims made)

Input¶

Input handling Status
Input System (new) Fully supported: keyboard, mouse, gamepad incl. UI navigation
Legacy Input Manager Supported: keyboard + mouse fully; gamepad right-stick look and triggers unavailable (the default InputManager.asset defines no axes for them and the package does not edit ProjectSettings)
Both Fully supported (verified)

Platforms¶

Platform Status
Windows / macOS / Linux standalone Fully supported (Mono and IL2CPP; IL2CPP verified with Managed Stripping Level = High)
WebGL Not supported in this version (capture-to-disk); known limitation, roadmap item
Mobile (touch UI) Untested / roadmap (no touch UI yet)
Consoles Untested / roadmap: the gamepad input layer is console-style, but no platform claims are made
